三、修正版數學式

1、灰階公式

Gray ≈ 0.299R + 0.587G + 0.114B
Gray = round_even((77R + 150G + 29B) / 256)
def gray_round_ties_to_even(r, g, b):
    s = 77*r + 150*g + 29*b
    q, rem = divmod(s, 256)
    if rem > 128 or (rem == 128 and (q & 1)):
        q += 1
    return q

2、Gaussian Filter

K = [[1,2,1],[2,4,2],[1,2,1]], sum(K)=16
G(x,y)=round_even(Σ K(i,j)·Gray(x+i,y+j) / 16)

3、Otsu Threshold

W_b(T)=Σ(i=0..T)h(i)/N，W_f(T)=Σ(i=T+1..255)h(i)/N
μ_b(T)=Σ(i=0..T)i·h(i)/Σ(i=0..T)h(i)，μ_f(T)=Σ(i=T+1..255)i·h(i)/Σ(i=T+1..255)h(i)
σ_B²(T)=W_b(T)W_f(T)(μ_b(T)-μ_f(T))²

4、Mask、CCL 與 BBox

m(x,y)=1 if G(x,y)>T；otherwise 0
visited 8-connected neighbors = {W, NW, N, NE}
BBox = (xmin, xmax, ymin, ymax)
Output = GREEN if pixel lies on bbox boundary；otherwise original RGB

五、轉成 Verilog 的學習重點

S_LOAD       : 讀取 ImgROM 原始 RGB
S_GRAY       : RGB 轉灰階
S_GAUSS_HIST : Gaussian filter 並建立 histogram
S_OTSU_SCAN  : 掃描 threshold，找最大 between-class variance
S_MASK_WRITE : 產生 binary mask 並做 auto-polarity
S_CCL_SCAN   : 8-connected CCL
S_BBOX_SCAN  : 計算 bounding box
S_DRAW_WRITE : 將 bbox 邊界畫成 32'h0000_FF00

Verilog 不是逐行翻譯 Python，而是把演算法拆成硬體可執行的狀態、暫存器與記憶體控制。每個 clock 只完成一部分工作，並透過 address counter 掃描整張 256×256 影像。
