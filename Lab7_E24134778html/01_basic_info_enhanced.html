本報告為 Introduction to VLSI CAD 課程之 Lab7：Draw Bounding Box Engine。作業目標為設計一個可於硬體中執行之影像處理流程，將 256×256 RGB 輸入影像經由灰階轉換、濾波、門檻分割、Connected Component Labeling 與 Bounding Box 計算後，輸出含有綠色邊界框之結果影像。本設計目前已於 ModelSim RTL simulation 中完成 P1 至 P4 四組測試，所有 pattern 之 error 數皆為 0，表示輸出結果與 golden result 在 pixel-level 上完全一致。

本頁不放完整 Verilog source code，只作為報告首頁與基本資料頁；詳細演算法、I/O、FSM 與驗證結果放在後續章節。
