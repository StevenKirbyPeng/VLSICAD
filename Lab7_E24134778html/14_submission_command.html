14. 十四、Submission Command

題目說明

本題要求將所有指定檔案壓縮成一個 .tar 檔並透過 Moodle 繳交。若設計使用額外檔案或子模組，也必須一併附上。

建議提交資料夾

Lab7_E24134778/
├── drawBbox.v
├── tb.sv
├── CYCLE_MAX.sv
├── drawBbox_syn.v              # synthesis 後補入
├── drawBbox_syn.sdf            # synthesis 後補入
├── DC.sdc
├── .synopsys_dc.setup
├── superlint.tcl
├── synthesis.tcl
├── Makefile
├── drawBbox_result_p1.bmp
├── drawBbox_result_p2.bmp
├── drawBbox_result_p3.bmp
├── drawBbox_result_p4.bmp
├── terminal_result_p1.log
├── terminal_result_p2.log
├── terminal_result_p3.log
├── terminal_result_p4.log
└── E24134778.pdf

Windows PowerShell 打包指令

tar -cvf Lab7_E24134778.tar Lab7_E24134778

Linux / EDA Server 打包指令

cd ~/Lab7_E24134778
tar -cvf ../Lab7_E24134778.tar .

若需先跑 synthesis

dc_shell
source synthesis.tcl
exit

若需檢查 area

cat area_rpt.txt
# 找 Total cell area

若需跑 post-synthesis simulation

vlog drawBbox_syn.v CYCLE_MAX.sv tb.sv
vsim work.tb -sdftyp /tb/u_drawBbox=drawBbox_syn.sdf
run -all
提交前檢查：不要放 .fsdb、.vcd、test_fig；若使用子模組一定要包含；若已合成，務必放入 drawBbox_syn.v 與 drawBbox_syn.sdf。

報告可貼結論

本作業最終應以 Lab7_E24134778.tar 形式提交至 Moodle。壓縮檔內需包含 RTL、testbench、synthesis files、result bmp、log files 與 final report，確保助教可直接解壓縮、編譯、模擬與驗證。
